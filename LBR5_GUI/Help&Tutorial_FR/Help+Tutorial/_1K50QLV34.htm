Coordonnées des noeuds

Dans l'onglet "Model" de la barre de Menu, cliquer sur "nodes", on retrouve la fenêtre suivante :


            

Pour modifier les coordonnées d'un noeud, il faut le sélectionner dans le tableau central puis cliquer sur modifier (Modify). Les coordonnées du noeud peuvent être changées dans les cases Y et Z.

Ensuite appuyer sur OK (la fenêtre se ferme) ou apply (la fenêtre reste ouverte) pour enregistrer le changement.
