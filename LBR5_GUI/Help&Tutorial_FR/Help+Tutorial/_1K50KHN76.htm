Paramètres d'optimisation

Pour effectuer une analyse avec optimisation, cliquer sur "Options" dans l'onglet "Project" de la barre de Menu. Ensuite choisir "Optimization", le tableau suivant apparaît:


                  

 

  • Optimisation : Choisir d'effectuer une analyse avec optimisation en cochant "Perform Optimisation"

 

  • Nombre d'itérations : choisir le nombre d'itérations maximum désiré lors du processus d'optimisation. Ce nombre s'étend de 1 à 20.

Pour une analyse rapide, on choisit un nombre d'itérations de 3 à 5.

Pour une anlyse plus poussée, on préfère un nombre de 10-15 itérations.

 

  • Fonction objectif : Choisir la fonction objectif de l'optimisation:

1° inertie maximum

2° poids minimum

3° Coût minimum avec le module simplifié

4° Coût minimum avec le module développé (disponible seulement avec des fiches CostCat)

 

Cliquer sur OK pour enregistrer les modifications.
