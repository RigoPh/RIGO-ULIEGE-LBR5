Variation de la fonction objectif

Pour obtenir la représentation de la variation de la fonction objectif au cours des itérations, cliquer sur "Objective Function Variation" de "Output Graphs" situé dans l'onglet "Analysis" de la barre des menus.


Le tableau suivant apparait :
 

 Snapshot : ce bouton permet d'enregistrer le graphe dans un fichier.

Print : ce bouton permet d'imprimer ce graphe.

Weight Objective Function  : Le titre du graphe .

               Weight Objective Function : Fonction objectif Poids

               Cost Objective Function : Fonction objectif coût.

Les valeurs données sont exprimées en Tonnes si la fonction objectif est le poids minimum, en euros si la fonction objectif est le coût minimum.
