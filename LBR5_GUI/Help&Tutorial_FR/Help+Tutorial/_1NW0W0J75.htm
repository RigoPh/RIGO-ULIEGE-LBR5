Caractérisation des éléments plaques

Pour introduire les caractéristiques des éléments plaques, procédez comme suit:

Modélisation

  1. Panels>Plate Scantlings

  2. Dans la fiche, remplacez les valeurs par défaut d'épaisseur de tôle (onglet Thickness), des dimensions des raidisseurs (onglet Stiffeners Set 1), des dimensions des cadres (onglet Frames Set 1) et des matériaux (onglet Materials) pour chacun des éléments plaques sur base des données présentées à Description du problème. Cliquez sur Apply pour valider et visualiser chacune de vos modifications. Cliquez ensuite sur OK.

    Remarque:

    • Dans cet exemple, ne tenez pas compte des autres onglets et des colonnes relatives aux épaisseurs de corrosion.

    • Afin d'accélérer l'introduction des données, vous pouvez séléctionner une ou plusieurs lignes d'un tableau, la ou les copier (bouton droit de votre souris, puis Copy) et la ou les coller sur d'autres lignes (bouton droit de votre souris, puis Paste).

Visualisation

Pour visualiser en détails les données introduites, procédez comme suit:

  1. View>Display Options

  2. Vous pouvez, p. ex., visualiser l'espacement introduit pour les cadres en sélectionnant la ligne Primary Frames et en sélectionnant Spacing dans le menu déroulant de la ligne Primary Frames Caption.


A ce stade, une sauvegarde du projet s'impose: File>Save
