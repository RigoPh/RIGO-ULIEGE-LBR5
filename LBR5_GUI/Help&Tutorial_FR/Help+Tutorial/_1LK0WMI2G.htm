Origine des axes

L'origine des axes peut être translatée. Pour ce faire, aller dans "move origin" situé dans l'onglet "Model" de la barre de Menu.
               

Introduire ensuite les coordonnées du point qui deviendra l'origine des axes.

Presser sur "OK" pour effectuer le changement.

Les coordonnées des autres points du modèle changent automatiquement.
