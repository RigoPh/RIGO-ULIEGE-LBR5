Ajout des éléments plaques

Avant d'introduire les éléments plaques dans le modèle du navire, il est important de vous rappeler la règle suivante:

Lorsque plusieurs éléments (plaques et/ou épontilles) sont connectés à un même noeud, il est nécessaire que ce noeud soit le noeud de départ d'au moins un élément et le noeud d'arrivée d'au moins un élément.

Procédez alors de la manière suivante:

  1. Model>Add Plate(s)

  2. A l'aide du curseur noir, sélectionnez d'abord le noeud de départ puis le noeud d'arrivée de chaque élément plaque.


A ce stade, une sauvegarde du projet s'impose: File>Save
