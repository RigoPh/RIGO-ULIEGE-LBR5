Entredistance des noeuds

L'entredistance entre deux noeuds peut être calculée. Cliquer sur "tools" dans la barre de Menu puis sur "Distance Between Nodes".

Sélectionner le noeud "origine" puis sélectionner le noeud "final", le tableau suivant apparaît reprenant la distance demandée.


                          

 

Pour sortir de cette commande sans sélcetionner les noeuds, cliquer sur "escape".
