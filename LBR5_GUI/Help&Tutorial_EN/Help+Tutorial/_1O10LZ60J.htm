Introduction des restrictions d'égalité

Pour introduire les restrictions d'égalité entre variables de conception, procédez de la manière suivante:

  1. Optimization>Equality Constraints

  2. Dans la boîte de dialogue, sélectionnez dans la colonne de gauche les éléments plaques présentant une ou plusieurs variables de conception, à choisir dans la colonne centrale, en relation avec l'élément plaque de la colonne de droite. Procédez ainsi pour toutes les restrictions d'égalité imposées dans Description du problème. Appuyez sur Apply pour valider.

  3. Pour visualiser les restrictions d'égalité introduites, appuyez sur la touche Show List de la boîte de dialogue. Appuyez sur Close pour quitter la boîte de dialogue.



A ce stade, une sauvegarde du projet s'impose: File>Save
