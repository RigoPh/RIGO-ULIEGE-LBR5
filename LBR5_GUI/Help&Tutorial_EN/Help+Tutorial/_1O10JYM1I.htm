Introduction des bornes

Pour sélectionner les variables de concepion et introduire leurs bornes, procédez comme suit:

  1. Optimization>Design variables

  2. Dans la boîte de dialogue, sélectionnez un élément plaque dans la colonne de gauche et introduisez les bornes de ses variables de conception sur base des données présentées à Description du problème. Appuyer sur Apply pour valider puis sur Close pour quitter la boîte de dialogue.

    Remarque:

    • Une variable de conception est active dans l'optimisation dès la définition de ses bornes.

    • Afin d'accélérer l'introduction des données, vous pouvez séléctionner une ou plusieurs variables de conception d'un élément plaque, la ou les copier (bouton droit de votre souris, puis Copy) et la ou les coller sur d'autres éléments plaques (bouton droit de votre souris, puis Paste).

    • Dans les données présentées à Description du problème, seules les bornes minimum ou maximum (espacements) sont données. Pour les autres valeurs, choisissez les de manière à avoir un intervalle de variation des variables de conception suffisamment large, tout en considérant des valeurs technologiquement admissibles.Par exemple,
      • Epaisseur maximum pour les bordés: 20 mm
      • Pour les cadres,
        • Hauteur d'âme maximum: 1000 mm
        • Epaisseur d'âme: 20 mm
        • Largeur de semelle: 500 mm
        • Ecartement minimum: 500 mm
      • Pour les lisses,
        • Hauteur d'âme maximum: 500 mm
        • Epaisseur d'âme: 20 mm
        • Largeur de semelle: 250 mm
        • Ecartement minimum: 100 mm
