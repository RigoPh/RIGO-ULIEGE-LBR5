Variation des variables de conception

Pour obtenir la représentation de la variation de chaque variable de conception au cours des itérations, cliquer sur "Design Variables Variation" de "Output Graphs" situé dans l'onglet "Analysis" de la barre des menus.


Le tableau suivant apparait :
                  

 

  • Snapshot : ce bouton permet d'enregistrer le graphe dans un fichier.
  • Print : ce bouton permet d'imprimer ce graphe.
  • Panel : Choisir le panneau pour lequel le graphe de la variation de ses variables est demandé.
  • Cocher les variables pour lesquelles le graphe est demandé. Le graphe à l'échelle apparait automatiquement.
  • Cocher Show Text pour que les valeurs des variables apparaissent.
