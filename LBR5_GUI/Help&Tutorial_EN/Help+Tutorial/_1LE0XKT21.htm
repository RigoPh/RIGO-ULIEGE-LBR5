Module coût développé

Cliquer sur "Optimization -> Cost -> CostCAt -> Wizard" dans la barre de menu. L’assistant CostCat apparaît. Parcourir toutes les étapes de l’assistant.

 

Etape 1: Paramètres globaux

Paramètres à introduire :

-         nombre d’abouts de montage ;

-         coût total de PMB (préparer – mettre – balancer) ;

-         temps tôlier ;

-         temps soudeur ;

-   nombre indices de double coque (si ce paramètre est laissé nul, il deviendra automatiquement 1 lorsque l’utilisateur va définir des éléments de double coque).  Le fractionnement résultant serait pour chaque panneau de double coque le rapport entre le nombre d’indices et le nombre total de panneaux de double coque.

 

Etape 2: Définition des types de panneau

Les types de panneau doivent être définis. Les types par default sont: bordé simple, épontille et panneau fictif. Le type épontille ne peut être changé. Si un panneau a une largeur plus petite ou égale a 2 mm, il est défini par default comme fictif. Les panneaux fictifs sont en fait des supports pour les conditions de bord et ne sont pas considérés dans le calcul du coût.

Pour définir un type de panneau, cliquer sur le bouton 'Ajouter Panneau' et choisir dans le menu contextuel un des types listés. La fenêtre assistant se ferme; sélectionner sur l'écran des panneaux pour les imposer le type choisi. Clique droite pour terminer la sélection et revenir à la fenêtre assistant.

Dans le cas du type double coque, sélectionner des panneaux pour former la coque intérieure, clique droite pour accepter la sélection, sélectionner les panneaux pour former la coque extérieure, de nouveau clique droite pour terminer et revenir à la fenêtre assistant.

 

Etape 3: Définition des nappes planes

 

Les nappes planes doivent être définies. Chaque panneau de type bordé simple ou de double coque est par default une nappe à part entière. L'utilisateur doit grouper les panneaux pour former des nappes planes en conformité avec les plans de conception.

Pour former une nappe plane, cliquer sur 'Ajouter Nappe'. La fenêtre assistant se ferme; sélectionner sur l'écran des panneaux pour les grouper dans une seule nappe. Clique droite pour terminer la sélection et revenir à la fenêtre assistant.

 

Etape 4.1: Opérations de pré-pré (voiles, transversaux)

Paramètres à introduire :

-          profilés sur âmes voiles (unités). Valeur par défaut : le nombre de lisses pour les bordés simples ou le nombre le plus réduit de lisses sur les doubles coques (inner ou outer) ;

-          soudures profilés sur âmes voiles (type – continue, discontinue). Valeur par défaut : soudure continue ;

-          goussets sur semelles de profilés (unités). Valeur par défaut : 0 ;

-          plats en buté (unités). Valeur par défaut : 0 ;

-          accostages de voiles (unités). Valeur par défaut : 0 ;

-          sections lisses (type – profilé, T synthétique). Valeur par défaut : profilé.

 

Etape 4.2: Opérations de pré-pré (serres, carlingues, hiloires)

Cette étape ne sera pas visible si l’utilisateur ne définit pas des panneaux de type carlingue.

 

Paramètres à introduire :

-          nombre de couples intermédiaires (unités). Valeur par défaut : 2 ;

-          habillage des carlingues (types – simple, double) Valeur par défaut : double ;

-   dimension pièce d’habillage (longueur (m) - < 1, > 1). Valeur par défaut : < 1.

 

Etape 5: Opérations de pré (fabrication de la nappe plane)

Paramètres à introduire :

-         nombre d’accostages (unités). Valeur par défaut : 0 ;

-         nombre d’abouts de nappe (unités). Valeur par défaut : 0;

-   soudure lisses (types – continue, discontinue). Valeur par défaut : continue.

 

Etape 6: Opérations de pré (assemblage)

Paramètres à introduire :

-         couples intermédiaires carlingues sur panneaux de nappe (unités). Valeur par défaut : 0 ;

-         type tapes (type – non-étanche-recouvrement, non-étanche-encastré,  étanches-recouvrement, étanche-encastré). Valeur par défaut : non-étanche-recouvrement;

-         section tapes (type – profilé, T synthétique).  Valeur par défaut : profilé ;  

-   accostages tôles individuelles de bouchain (unités). Valeur par défaut: 0. L’accostage du bouchain avec la tôle de fond (outer) est implicite,  il ne va pas être pris en compte par l’utilisateur. 

 

 

Etape 7: Opérations de pré-montage - montage

Paramètres à introduire :

-          contacts des barrots sur cloisons (unités). Valeur par défaut : 0 ;

-          jonctions barrot / hiloire (unités). Valeur par défaut : 0.

Cliquer sur "Optimization -> Cost -> CostCAt -> Workshop Coefficients" dans la barre de menu.

 Compléter le tableau avec les coefficients d'accessibilité et d'atelier du chantier.

 

Cliquer sur "Optimization -> Cost -> CostCAt -> Tables" dans la barre de menu.

Les bases de données du module CostCat constituées suite au parcours de l'assistant de configuration et du tableau de coefficients d'accessibilité et d'atelier, peuvent être visualisés et edités dans la fenêtre au-dessus.
