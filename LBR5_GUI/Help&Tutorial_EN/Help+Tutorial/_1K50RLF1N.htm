Analyse simple

Pour lancer l'analyse d'un projet, cliquer sur "Solve" dans l'onglet "Analysis" de la barre de Menu.

Pour lancer plusieurs projets les uns à la suite des autres sans intervenir, aller dans le chapitre "Analyse Multiple".

La fenêtre DOS suivante apparaît :

 

Lorsque l'analyse est terminée, cette fenêtre disparait et les fichiers résultats sont disponibles.
