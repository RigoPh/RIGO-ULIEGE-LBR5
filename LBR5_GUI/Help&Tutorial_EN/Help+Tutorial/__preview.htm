Graphical user interface

Presentation of the main window of the graphic interface:


1. La zone menu est située au dessus de l'écran et comporte différents onglets :

  • File : Ouvrir, fermer, créer, enregistrer, importer un fichier, consulter les propriétés du fichier, mettre en page, imprimer, envoyer un fichier, sortir de l'interface.
  • Edit : Annuler la dernière commande, couper, copier, coller.
  • View : Rafraîchir le dessin lors de changements, choisir l'affichage du dessin du modèle, ouvrir une page web, afficher les barres d'outil ou de message. 
  • Project : Introduire le titre du projet qui sera affiché dans les fichiers données et résultats. Choisir différentes options de calcul :l'optimisation, la prise en compte du poids propre...
  • Model : Entrer les coordonnées des noeuds, dessiner les panneaux, poutres et doubles coques, diviser les panneaux, inverser le sens de parcours du panneau, scinder la double coque en deux simples coques.
  • Panels : Entrer les dimensions des panneaux, poutres et doubles coques et leurs raidisseurs, entrer les conditions d'appui. 
  • Loads : Enumérer les cas de chargement, entrer les valeurs des charges. 
  • Optimization : Choisir le module coût adapté, les variables de dimensionnement, les restrictions structurelles, géométriques et d'égalité, la restriction sur la position du centre de gravité.
  • Tools : Calculer la distance entre les noeuds, vérifier la conformité du modèle, photographier l'écran.
  • Analysis: Résoudre, résoudre plusieurs projets, fichiers résultats et graphes.
  • Window :Afficher en cascade, alligner horizontalement, alligner verticalement, réarranger les icones.
  • Help: 

2. La zone Outil est située au dessus de l'écran et comporte différents onglets :

  • New :       Créer un nouveau projet 
  • Open :      Ouvrir un fichier existant
  • Save :      Sauvegarder rapidement
  • Zoom in :          Agrandir
  • Zoom out :       Diminuer
  • Zoom win :       Remplir la fenêtre avec la sélection
  • Zoom full :        Retourner à la taille initiale
  • Pan :              Capture d'un élément

 

3. La zone message est située au dessous de l'écran et comporte deux espaces :

  • L'écran désignant le mode actuel (normal, zoom....)

  • L'écran désignant le cas de charge actuel
