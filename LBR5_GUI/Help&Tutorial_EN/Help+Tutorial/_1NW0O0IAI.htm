Introduction du nombre de termes de Fourier

Pour introduire le nombre de termes de Fourier à employer dans l'analyse structurelle, procédez comme suit:

  1. Project>Options>Fourier Series

  2. Dans cet exemple, où le chargement est constant sur la longueur du navire, seuls les termes impairs seront pris en compte lors de l'analyse structurelle. On considère 7 termes impairs de Fourier dans cet exemple. Pour ce faire, utilisez le menu déroulant dans la boîte de dialogue et sélectionnez -7. Cliquez sur OK pour valider.
