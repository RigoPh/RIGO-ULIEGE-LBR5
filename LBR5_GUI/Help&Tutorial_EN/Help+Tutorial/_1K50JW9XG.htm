Restrictions globales

Les restrictions sur la position du centre de gravité peuvent être déterminées dans l'onglet "Gravity Center" du chapitre "optimization" de la barre de Menu. Le tableau suivant apparaît :


                                

Restrictions sur le centre de gravité :

  • Choisir si la position verticale du centre de gravité est limitée ou non. Soit la position n'est pas limitée (No Active Limits), soit le Centre de gravité ne peut descendre sous une certaine position (Lower Limit Active), soit il ne peut monter au delà d'une position (Upper Limit Active), Soit la position du centre de gravité est comprise dans un intervale (Both Limits Active).
  • Si la position du centre de gravité est limitée, introduire la limite correspondante. Celle-ci est mesurée à partir de l'origine des axes.

Restriction sur l’inertie :

  • Cocher "Minimum Inertia" pour activer la restriction. Introduire la limite minimum de l’inertie.

Restriction sur le coût :

  • Cocher "Minimum Cost" pour activer la restriction. Introduire la limite minimum du coût.

Restriction sur le module de résistance :

  • Cocher "Minimum Section Modulus" pour activer la restriction. Introduire la limite minimum du module de résistance. Choisir un panneau pour déterminer la distance verticale maximum (calculé dans le centre du panneau) par rapport au centre de gravité de la section.

 

Presser le bouton "OK" pour enregistrer les modifications.
