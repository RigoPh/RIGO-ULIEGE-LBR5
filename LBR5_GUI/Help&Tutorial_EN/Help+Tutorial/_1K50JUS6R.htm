Restrictions d'égalité

Les restrictions d'égalité peuvent être déterminées dans l'onglet "equality constraints" du chapitre "optimization" de la barre des menus. Le tableau suivant apparait :
         

Les restrictions d'égalité consistent à choisir des relations entre les deux mêmes variables de panneaux différents.

  • Choisir le panneau dépendant (Dependent Variables). Les variables de ce panneau varieront pour satisfaire la restriction d'égalité.
  • Choisir le panneau indépendant (Independent Variables). Les variables de ce panneau ne varieront pas pour satisfaire la restriction d'égalité.

 Les variables en noir, sont les variables communes possibles aux deux panneaux.

  • Choisir la ou les variable(s) qui sera commune aux deux panneaux. Elle devient soulignée.

      

  • Indiquer le ratio désiré puis presser le bouton "=". Le nombre de restrictions sélectionnées est indiqué.

     

Pour voir la liste des restrictions choisies, cliquer sur "show list". Le détail des restrictions est indiqué.

      

Presser le bouton "numeric view" pour obtenir l'expression des restrictions comme elles apparaissent dans le fichier input de LBR5.

Pour éliminer une restriction, cliquer sur "show list", puis à partir de la liste, séléctionner la restriction en question et cliquer sur le bouton "Remove Selected".

Attention: Un panneau dépendant pour une restriction sur une variable, ne peut pas être indépendant (et donc référence) pour une restriction sur cette même variable.

Lorsque toutes les restrictions sont introduites, cliquer sur "apply" pour enregistrer la liste.
