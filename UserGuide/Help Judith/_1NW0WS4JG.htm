Définition des cas de charge

Pour définir les deux cas de charge considérés dans cet exemple, procédez comme suit:

  1. Loads>Load Cases

  2. Dans la boîte de dialogue, cliquez sur Add pour ajouter un cas de charge. Vous pouvez ensuite remplacez le nom par défaut Load Case 1 par un autre, p. ex., Hogging. Pour activer les cas de charge, assurez vous que la valeur donnée à la colonne State est ON.
