plaques

Pour déterminer l'échantillonnage des plaques, cliquer sur "panels" dans la barre de Menu puis sur "plate scantlings". La fenêtre suivante apparaît:

        

Plusieurs onglets permettent de choisir les dimensions des plaques raidies :

  

  • Géometrie : L'onglet "Geometry" reprend les dimensions principales de la plaque : Sa largeur en mètres (width), L'angle d'inclinaison (Angle), le noeud de départ (In node) et d'arrivée (Out node). Ces informations sont automatiquement reprises lors de la création/édition de la géométrie.

L'angle est déterminé à partir de l'horizontale à droite du noeud de départ, positif dans le sens horlogique.

     

 

  • Epaisseurs : L'onglet "Thickness" reprend l'épaisseur nette (Net thickness en mm) et l'épaisseur de corrosion (Corrosion thickness en mm). L'épaisseur de corrosion est l'épaisseur à ajouter à l'épaisseur nette pour obtenir l'épaisseur réelle (brute) de la tôle. Cette épaisseur de corrosion ne doit être appliquée qu'aux tôles exposées à l'eau de mer ou à un chargement liquide.

                             

 

  • Raidisseurs Principaux : L'onglet "Stiffeners Set 1" permet de choisir leur distribution (EE1 ou EE2), les dimensions de l'âme : Hauteur (Web Height en mm) et épaisseur (Web Thickness en mm), les dimensions de la semelle : Largeur (Flange Width en mm) et épaisseur (Flange Thickness en mm), l'espacement entre les raidisseurs (Spacing en mm), l'épaisseur de Corrosion à appliquer (Corrosion Thickness en mm), ainsi que le côté de la tôle où les raidisseurs sont positionnés.    

La distribution des raidisseurs peut être déterminée de 2 façons :

Soit EE1 : Le premier espacement est compté comme entier à partir du noeud de départ.

Soit EE2 : Le premier espacement vaut la moitié de l'espacement entre raidisseurs à partir du noeud de départ.

Le côté où se trouve les raidisseurs est déterminé suivant le sens de parcours du panneau. Il ne peut y avoir des raidisseurs sur les 2 côtés du panneau.

  • Cadres Principaux : Ils (Frames set 1) sont déterminés comme les raidisseurs principaux. La distribution est automatiquement de type EE2.

 

  • Raidisseurs Secondaires : Ils (Stiffeners Set 2) sont déterminés comme les raidisseurs principaux. La distribution est automatiquement de type EE2.

 

  • Cadres Secondaires :Ils (Frames set 2) sont déterminés comme les raidisseurs principaux. La distribution est automatiquement de type EE2.

 

  • Traverses : Le nombre maximum de traverses (Girders) par panneau est de 10. Les traverses sont définies par leurs positions par rapport au point de départ du panneau (Distance From In Node en m), leurs hauteurs d'âme (Web Height en mm), leurs épaisseurs d'âme (Web Thickness en mm), leurs largeurs de semelle (Flange Width en mm), leurs épaisseurs de semelle (Flange Thickness en mm). En sélectionnant le panneau auquel les traverses sont rapportées, la longueur de ce panneau apparait (Panel Length en m) et permet de placer les traverses à une position inclue dans le panneau. Le côté du panneau où se trouvent les traverses est déterminé suivant le sens de parcours du panneau.

 

  • Matériaux : Les caractéristiques des matériaux peuvent être introduites dans l'onglet "Materials". Les caractéristiques sont les suivantes : Module de Young (Young Modulus en N/mm²), Coefficient de poisson (Poisson Coefficient), Limite élastique (Yield Stress en N/mm²), Contrainte admissible utilisée dans le set de restriction de Hughes (Allowable Stress en N/mm²), Poids Spécifique (Specific Weight en t/m³). Par défaut, le matériau est l'acier.

  • Participation : Le niveau de participation du panneau intervenant dans le moment de flexion global autour de l'axe horizontal se situe entre 0 et 1. Ceci n'est pas envisageable si un moment de flexion global vertical est appliqué. 

 

Presser apply pour visionner les modifications tout en gardant la fenêtre ouverte, OK pour que les données soient enregistrées et la fenêtre fermée.
