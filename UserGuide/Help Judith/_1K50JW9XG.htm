Restrictions sur le C.G.

Les restrictions sur la position du centre de gravité peuvent être déterminées dans l'onglet "Gravity Center" du chapitre "optimization" de la barre de Menu. Le tableau suivant apparaît :
                                

  • Choisir si la position verticale du centre de garvité est limitée ou non. Soit la position n'est pas limitée (No Active Limits), soit le Centre de gravité ne peut descendre sous une certaine position (Lower Limit Active), soit il ne peut monter au delà d'une position (Upper Limit Active), Soit la position du centre de gravité est comprise dans un intervale (Both Limits Active).
  • Si la position du centre de gravité est limitée, introduire la limite correspondante. Celle-ci est mesurée à partir de l'origine des axes.

Presser le bouton "OK" pour enregistrer les modifications.
