Propriétés semblables

Si certains panneaux possèdent les mêmes propriétés, il suffit de définir les propriétés d'un panneau et de désigner le ou les autre(s) panneau(x) semblables.

Choisir "Match properties" dans l'onglet "Model" de la barre de Menu. Sélectionner le panneau référence.
Sélectionner avec le pinceau, le ou les panneaux dont les propriétés seront modifiées pour être équivalentes au panneau de référence.

Pour sortir de la commande "propriétés semblables", presser escape.

Une fois sorti de cette commande, les liens de similarité entre panneaux sont coupés et les propriétés de chaque panneau redeviennent indépendantes.

Actuellement les propriétés dont il est questions précédemment ne concernent que la géométrie.
