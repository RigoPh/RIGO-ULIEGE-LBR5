Noeuds

Dans l'onglet "model" de la barre Menu, les noeuds peuvent être déterminés.

              

Pour entrer des noeuds, entrer les coordonnées dans les cases Y et Z puis cliquer sur "add".

Pour éliminer les noeuds sélectionnés, appuyer sur "Remove".

Les distances entre noeuds dans le repère x, y , z sont exprimées en mètres.

Pour les autres fonctions voir le chapitre "coordonnées des noeuds" dans "Editer la Géométrie".

Pour que les noeuds soient enregistrés après création ou modification, cliquer sur OK (la fenêtre se ferme automatiquement) ou sur APPLY (la fenêtre reste ouverte).
