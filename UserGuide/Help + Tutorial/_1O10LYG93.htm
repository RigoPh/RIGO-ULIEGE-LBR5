Introduction des restrictions géométriques

Pour introduire les restrictictions géométriques, procédez de la manière suivante:

  1. Optimization>Geometrical Constraints

  2. Dans la boîte de dialogue, sélectionnez un élément plaque dans la colonne de gauche. Appuyer sur la touche References pour obtenir le récapitulatif des restrictions géométriques disponibles. Dans l'exemple proposé, les prescriptions technologiques de Hughes sont considérées. Cliquez donc sur l'onglet Sets et cochez la case relative à Set of Hughes (for T members). Procédez ainsi pour chaque élément plaque. Cliquez ensuite sur Apply pour valider puis sur Close pour quitter la boîte de dialogue.

    Remarque:


    • Afin d'accélérer l'introduction des données, vous pouvez séléctionner une ou plusieurs restrictions, la ou les copier (bouton droit de votre souris, puis Copy) et la ou les coller sur d'autres éléments plaques (bouton droit de votre souris, puis Paste).


A ce stade, une sauvegarde du projet s'impose: File>Save
