Activation de l'optimisation

Afin de réaliser l'optimisation dimensionnelle, sur base du poids, du maître couple proposé, procédez comme suit:

  1. Project>Options>Optimization

  2. Dans la boîte de dialogue, cochez la case Perform Optimization, et Minimal Weight dans les types d'optimisation proposés. A l'aide du menu déroulant, choisissez ensuite le nombre d'itérations d'optimisation, p. ex., 10. Cliquez sur OK pour valider.


A ce stade, une première sauvegarde du projet s'impose:

  1. File>Save As

  2. Dans la boîte de dialogue, choisissez l'emplacement où vous voulez enregister votre projet et un nom pour celui-ci, p. ex. Tutorial(opti). Cliquez ensuite sur Enregistrer.
