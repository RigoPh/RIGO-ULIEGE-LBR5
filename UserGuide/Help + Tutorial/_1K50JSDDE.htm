Variables de dimmensionnement

Dans "Optimization" de la barre de Menu, les variables de dimmensionnement à optimiser peuvent être choisies dans  l'onglet "Design Variables". Le tableau suivant apparaît :
      

 Panels : Choisir les panneaux pour lesquels les variables doivent être optimisées.

Les variables :

Il y a 9 variables de dimensionnement possibles pour les plaques :  

1. Epaisseur des plaques : Plate Thickness   

2. Hauteur de l'âme des cadres transversaux : Frames Web Height

3. Epaisseur de l'âme des cadres transversaux : Frames Web Thickness

4. Largeur de la semelle des cadres transversaux : Frames Flange Wide

5. Espacement des cadres transversaux : Frames Spacing

6. Hauteur de l'âme des raidisseurs longitudinaux : Stiffeners Web Height

7. Epaisseur de l'âme des raidisseurs longitudinaux : Stiffeners Web Thickness

8. Largeur de la semelle des raidisseurs longitudinaux : Stiffeners Flange Wide

9. Espacement des raidisseurs longitudinaux : Stiffeners Spacing

 

Pour les épontilles, seules certaines variables peuvent être optimisées :

1. Epaisseur de la paroi mince (reprise sous Plate Thickness). Cette variable n'est disponible que dans le cas des sections carrées ou circulaires.

2. Hauteur de l'âme pour une section en double Té, Diamètre extérieur pour une section à paroi mince circulaire, Côté extérieur pour une section à paroi mince carrée. (reprise sous Frames Web Height)

3. Epaisseur de l'âme du double Té (reprise sous Frames Web Thickness). Cette variable n'est disponible que dans le cas d'une section en double Té.

4. Largeur de la semelle du double Té (reprise sous Frames Flange Wide). Cette variable n'est disponible que dans le cas d'une section en double Té.

5. Espacement des poutres : Frames Spacing

 

Les valeurs introduites lors de la définition de la géométrie sont reprises dans la case centrale (Xi en millimètres).

 Valeur minimale (XIMINi en millimètres) : Introduire la valeur minimale de chaque variable de dimensionnement.

Valeur maximale (XIMAXi en millimètres) : Introduire la valeur maximale de chaque variable de dimensionnement.

Ces deux dernières valeurs permettent de déterminer l'intervalle de variation de la variable de dimensionnement lors de l'optimisation. Lorsque la variable vaut une de ces deux valeurs extrèmes, elle devient rouge.

 

Cliquer sur Apply pour enregistrer les modifications.
